Leaderboard Database Setup

‚ö†Ô∏è Important Note

The current leaderboard shows only demo data because it's stored locally on each user's device.

For a REAL leaderboard that tracks all users globally, you need a backend database.

Option 1: Simple (Firebase)

Free, easy to set up, good for small apps

// 1. Create account at https://firebase.google.com
// 2. Add this to your code:

import { initializeApp } from "firebase/app";
import { getDatabase, ref, set, get } from "firebase/database";

const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_AUTH_DOMAIN",
  databaseURL: "YOUR_DATABASE_URL",
  projectId: "YOUR_PROJECT_ID"
};

const app = initializeApp(firebaseConfig);
const db = getDatabase(app);

// Save user stats
async function updateLeaderboard(userId, userName, weeklyAds) {
  await set(ref(db, 'leaderboard/' + userId), {
    name: userName,
    ads: weeklyAds,
    timestamp: Date.now()
  });
}

More powerful, scalable, full control
